# Multiplikationstabellen

Det här projektet är nu ombyggt till en statisk webbapp som fungerar direkt på GitHub Pages.

## Filer

- `index.html` innehåller appens struktur.
- `styles.css` innehåller layout, färger och responsiv design.
- `script.js` hanterar frågegenerator, poäng, statistik och historik.

## Publicera på GitHub Pages

1. Pusha repot till GitHub.
2. Öppna repository settings.
3. Gå till **Pages**.
4. Välj deploy från din huvudbranch och mappen `/ (root)`.
5. Spara. GitHub publicerar då appen från `index.html`.

## Lokal körning

Öppna `index.html` i webbläsaren så fungerar appen utan server.
